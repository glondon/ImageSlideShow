Basic Image Slide Show
How To Use The basic image slide show plugin is a simple image slide show that will take the images that you provide and turn them into a slide show. The plugin will also accept links by wrapping your img in a link tag. In order to use the plugin simply add your images into a element and then call the plugin in a script to that element as shown below. 

<!DOCTYPE HTML>

<html>

<head>

	<meta charset="utf-8">

	<title>Untitled Document</title>

<script>

$(document).ready(function() {

    $('#imageslider').ImageSlide();

});

</script>

</head>



<body>



<div id="imageslider">

    <a href="http://www.google.com"><img src="images/13251689153_34ccfbdaaa_o.jpg"></a>

    <img src="images/daytime-city-view.jpg">

    <a href="http://www.msn.com"><img src="images/DSC_0014.jpg"></a>

</div>

</body>

</html>
Options
speed This is the speed in milli-seconds which is 1000 per second.
arrows By adding false would remove the arrows from the slide show
radio By adding false would remove the radio buttons from the botton the slide show
